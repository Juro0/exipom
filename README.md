#  EXIPOM-it

## descrizione
exipom nasce come un semplice timer per la tecnica pomodoro, presenta tre modalità specifice predefinite, differenti per tempo. Dalla schermata impostazioni, oltre ad una personalizzazione delle modalità ed alcune preferenze, si possono modificare i vari colori del sito. Tutta la schermata impostazioni è salvata in LocalStorage e resettabile in qualsiasi momento.

# EXIPOM [en]

## description
exipom was born as a simple timer for the pomodoro technique, it has three specific predefined modes, different in time. From the settings screen, in addition to a customization of the modes and some preferences, you can change the various colors of the site. The entire settings screen is saved in LocalStorage and can be reset at any time.

## state
the project is currently under development, but accessible in version v1.0.0 .

## opensource
the project is open source and modifiable by anyone through forks and subsequent pull requests.
those who create a pull request that is subsequently accepted will be mentioned in the [authors](#author) section of the README.md.
